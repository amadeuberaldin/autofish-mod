# 🎣 AutoFish Mod (Fabric)

A lightweight client-side Minecraft mod that automates fishing with smart timing, real-time statistics, and an in-game configuration menu.

---

## ✨ Features

- 🎯 Automatic fishing with natural timing
- 🧠 Smart detection of fish bites
- 📊 Real-time HUD with statistics
- ⚙️ In-game configuration menu
- 💾 Persistent settings (JSON)
- 🔒 Client-side only (no server required)

---

## 📊 HUD Information

- Fish caught
- Treasure items
- Junk items
- Session time

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| R   | Toggle AutoFish |
| H   | Toggle HUD |
| M   | Toggle messages |
| O   | Open config menu |

---

## ⚙️ Configuration

Settings are stored in:

config/autofish.json


You can adjust:

- HUD visibility
- Message display
- Recast delay (min/max)

---

## 🛠️ Requirements

- Minecraft 1.21.11
- Fabric Loader
- Fabric API

---

## 🚀 Build

```bash
./gradlew build

Output:

build/libs/

---

📦 Installation
1. Install Fabric Loader
2. Put the .jar file in your mods folder
3. Launch Minecraft
